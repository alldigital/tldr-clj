# tldr-clj

![Screenshot](screenshot.png)

## Installation

Download from <https://github.com/iovxw/tldr-clj/releases/tag/v1.0.0>.

## Usage

    $ alias tldr="java -jar /path/to/tldr.jar"
    $ tldr -h
    Usage: tldr [options] command

    Options:
      -u, --update  Update cache
      -h, --help
